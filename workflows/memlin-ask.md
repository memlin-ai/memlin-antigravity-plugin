---
name: memlin-ask
description: Ask the Memlin workspace a natural-language question and get a cited answer from the team's shared memory, skills, goals, and schemas.
---

Run `memlin ask "<question>"` in the terminal, passing the user's question
(the `memlin` CLI is installed by this plugin — see the README/installer). Relay
the answer with its citations (path + version). For a raw search instead of a
synthesized answer, call the `memlin_search` MCP tool.
