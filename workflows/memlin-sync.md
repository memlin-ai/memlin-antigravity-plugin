---
name: memlin-sync
description: Full bidirectional Memlin sync — pull the latest team/project memory and skills, then push local changes.
---

Run `memlin sync` in the terminal and relay the result (the `memlin` CLI is
installed by this plugin — see the README/installer).
This pulls the latest memory/skills/goals for the bound project, then
pushes any local changes. Report what was pulled and pushed. For one-way sync
use `memlin pull` or `memlin push`.
