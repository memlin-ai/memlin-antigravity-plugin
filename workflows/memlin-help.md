---
name: memlin-help
description: Show the categorized list of every Memlin command (discovery, sync, plans, actions, audit, coordination, setup) — what each does in one line.
---

Run `memlin help` in the terminal — the dispatcher prints a host-aware cheat
sheet grouped by purpose. Relay the output verbatim. If the `memlin` CLI is
not on PATH, tell the user to run `install.sh` (bundled with this plugin) or
sign in via `memlin login` first.
