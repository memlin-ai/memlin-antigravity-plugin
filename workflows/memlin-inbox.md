---
name: memlin-inbox
description: Review the Memlin scribe inbox — list proposed decisions/memories/skills and accept or reject them.
---

Run `memlin inbox` in the terminal to list pending proposals (the `memlin` CLI
is installed by this plugin — see the README/installer). To act on one, run
`memlin inbox accept <id>` or `memlin inbox reject <id>`. Relay the list and
confirm each action taken.
